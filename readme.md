
# Medication Adherence Prediction & Forecasting

A brief description of what this project does and who it's for
A machine learning and deep learning project that predicts patient medication adherence and forecasts future adherence trends using classification models and LSTM networks.

## Features
📌 Binary classification of patient adherence
📈 Time-series forecasting using LSTM
🔄 Multi-dataset integration (Hypertension & Diabetes)
📊 Data visualization and exploratory analysis
⚙️ End-to-end ML pipeline


## Tech Stack

Python

Pandas, NumPy

Scikit-learn

XGBoost

TensorFlow / Keras

Matplotlib, Seaborn
